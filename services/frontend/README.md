# Frontend Service

React-based frontend application for FinTech Platform.

## Development
```bash
npm install
npm run dev
```

## Testing
```bash
npm run test:unit
npm run test:e2e
npm run test:coverage
```

## Building
```bash
npm run build
```

## Docker
```bash
docker build -t fintech-frontend .
docker run -p 3000:80 fintech-frontend
```
